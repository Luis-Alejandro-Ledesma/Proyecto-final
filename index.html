Derivación Sucesiva en Python

Para derivar funciones múltiples veces en Python usamos SymPy.

from sympy import symbols, diff

x = symbols('x')
f = x**4 + 3*x**3 - 2*x + 7

# Derivadas sucesivas
f1 = diff(f, x)       # Primera derivada
f2 = diff(f, x, 2)    # Segunda derivada
f3 = diff(f, x, 3)    # Tercera derivada

print(f1)
print(f2)
print(f3)

Teorema del Límite Central en Python

El Teorema del Límite Central establece que la media de muestras grandes de una población tiende a distribuirse normalmente.

import numpy as np
import matplotlib.pyplot as plt

# Generamos una población no normal
poblacion = np.random.exponential(scale=2.0, size=100000)

medias = []
for _ in range(1000):
    muestra = np.random.choice(poblacion, size=30)
    medias.append(np.mean(muestra))

# Gráfica de las medias muestrales
plt.hist(medias, bins=30, color='#ffab73', edgecolor='black')
plt.title("Distribución de medias (CLT)")
plt.xlabel("Media muestral")
plt.ylabel("Frecuencia")
plt.show()
